### Before we begin.

Run the following command in your terminal to test that you have installed dart properly.

```bash
dart --version

> Dart SDK version: 3.4.4 (stable) (None) on "linux_x64"
```

### Set up H4.

Run the command below to get started.

```bash title="activate"
dart pub global activate h4
```

Now, let's bootstrap your first app.

```bash title="init"
dart pub global run h4:init
```

After answering the prompts, the directory containing your app will have a file structure like
this..

<Files>
	<Folder name='.dart_tool'>
		<File name='package_config.json' />
	</Folder>
	<Folder name='bin'>

		/>
	</Folder>
	<Folder name='test' />
	<File name='.gitignore' />
	<File name='analysis_options.yaml' />
	<File name='CHANGELOG.md' />
	<File name='pubspec.lock' />
	<File name='pubspec.yaml' />
	<File name='README.MD' />
</Files>

Navigate to index.dart. Some code is already there.

```dart title="index.dart"
import 'package:h4/create.dart';

void main() async {
  var app = createApp(port: 5174);
  var router = createRouter();
  app.use(router);
// [!code highlight:2]
  router.get("/", (event) => 'Hello world!')
}
```

You're all set! Let's run our server in dev mode.

```bash
dart pub global h4:start --dev
```

Visit - http://localhost:5173 with your browser and voila! **Hello world!**
