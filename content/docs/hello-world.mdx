### Pre-requisites.

You must have the dart SDK installed on your computer.

<Card
	title='Installing dart'

/>

<br />

Run the following command in your terminal to test your installation.

```bash
dart --version

> Dart SDK version: 3.4.4 (stable) (None) on "linux_x64"
```

### Set up H4.

Run the following commands

```bash
dart pub global activate h4
```

Let's bootstrap your first app by running the command below in your terminal.

```bash
dart pub global run h4:init
```

After answering the prompts, you will have a directory either in the current path or
`/your_app_name` that looks like this..

Navigate to index.dart.

```dart title="index.dart" icon="true"
void main() async {
  var app = createApp(port: 5174);
  var router = createRouter();
  app.use(router);
// [!code highlight:2]
  router.get("/", (event) => 'Hello world')
}
```

Yay! The app is ready to go! Let's run our server.

```bash
dart pub global h4:start --dev
```

Visit localhost:5173 with your browser and voila! - `Hello world`.

You've just taken your first steps.
